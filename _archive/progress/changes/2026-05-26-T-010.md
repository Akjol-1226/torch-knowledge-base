# T-010 — core/vlm/schemas + parser

## 改了什么
- 新增 `backend/app/core/vlm/__init__.py`、`schemas.py`、`parser.py`
- 新增 `backend/tests/unit/test_vlm_parser.py`（5 个单测）

## 为什么改
入库流水线第一步：把 VLM 输出的 `<notsure>` 标记从 raw_text 中抽出来，方便人工 review。schemas 定义跨模块共享的 `NotsureSpan` 和 `VLMResponse`。

## 思路
- 标记格式定为 XML 风格 `<notsure>...</notsure>`（plan OQ-1 默认答案）
- 嵌套时取最外层用栈式扫描而非正则贪婪/非贪婪（正则在嵌套上不够稳）
- 解析返回 `NotsureSpan(start, end, text)`，end/start 是字符偏移，方便前端直接 highlight

## 验收
- `uv run pytest tests/unit/test_vlm_parser.py -v` → 5 passed
- 覆盖：empty / 单段 / 多段 / 嵌套 / 多行
