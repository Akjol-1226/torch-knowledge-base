# T-015 — modules/ingest/service + router

## 改了什么
- 新增 `backend/app/modules/ingest/schemas.py`（UploadResponse / TaskItem）
- 新增 `backend/app/modules/ingest/service.py`（IngestService.upload_document / list_tasks）
- 新增 `backend/app/modules/ingest/router.py`（POST /ingest/upload, GET /ingest/tasks）
- 改 `backend/app/main.py`：include ingest_router
- 新增 `backend/tests/integration/test_ingest_router.py`（2 个集成测试）
- 改 `backend/tests/conftest.py`：顶部显式 import ingest.models（见 lessons L-004）

## 为什么改
打通第一个真实业务 endpoint：上传文件 → 哈希去重 → 建 Document + IngestTask → 返回 task_id。任务消费（langgraph）不在本 task 范围。

## 思路
- service 用 sha256 去重（同文件多次上传共用 Document）
- `page_count=0` 暂存（真 PDF 解析在 ADR-002 后真实实现）
- router 走 DI（`Depends(get_db)`），符合 AGENTS.md R-3
- 集成测试用 `_engine = db_engine` patch 全局，避免起真实数据库连接

## 验收
- `uv run pytest tests/integration/test_ingest_router.py -v` → 2 passed
