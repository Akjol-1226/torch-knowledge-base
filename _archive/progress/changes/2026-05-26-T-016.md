# T-016 — modules/chat 占位 router + modules/auth 占位依赖

## 改了什么
- 新增 `backend/app/modules/chat/{__init__,router}.py`（/chat/echo 端点）
- 新增 `backend/app/modules/auth/{__init__,dependencies}.py`（CurrentUser + get_current_user）
- 改 `backend/app/main.py`：include chat_router
- 新增 `backend/tests/integration/test_chat_echo.py`（1 个集成测试）

## 为什么改
- chat 模块占位：让前端可以在 server skeleton 阶段就有可调通的 endpoint，避免「等真实 RAG 链路」阻塞前端联调
- auth 模块占位：让其他模块需要鉴权时可以 `Depends(get_current_user)`，先返回 anonymous 不挡道

## 思路
- chat 用最简 echo（POST 收 `{"message":"..."}` 回 `{"echo":"..."}`），后续真实 SSE 流式聊天会重写
- auth 完全没鉴权逻辑，只暴露 CurrentUser pydantic model 和 get_current_user 依赖
