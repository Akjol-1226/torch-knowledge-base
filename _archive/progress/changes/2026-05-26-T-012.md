# T-012 — modules/ingest/models (Document/Page/IngestTask)

## 改了什么
- 新增 `backend/app/modules/{__init__,ingest/__init__,ingest/models}.py`
- 新增 `backend/tests/unit/test_ingest_models.py`（3 个单测）
- 改 `backend/tests/conftest.py` 给 `db_session` 加 `expire_on_commit=False`（见 lessons L-003）

## 为什么改
入库流水线的核心数据模型。Document = 上传的文件，Page = 单页 + VLM 输出，IngestTask = 异步任务状态。

## 思路
- `Document.sha256` 唯一索引（去重）
- `Page.document_id` foreign key + index
- `IngestTask.status` 默认 "pending"，有 index 方便 list_tasks 过滤
- 用 `datetime.now(timezone.utc)` 替代 plan 中的 `datetime.utcnow()`（后者 3.12 deprecated；ruff 也会 warning）
