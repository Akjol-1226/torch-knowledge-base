# T-006 — 项目初始化（uv + pyproject + .env.example）

## 改了什么
- 创建 `backend/` 目录
- 新增：`backend/pyproject.toml`、`backend/.env.example`、`backend/.gitignore`、`backend/.python-version`、`backend/README.md`
- 比 plan 增补一项依赖：`aiosqlite>=0.20.0`（DATABASE_URL 用 `sqlite+aiosqlite` driver，必须显式装这个 driver，否则 T-009 测试会失败）

## 为什么改
按 plan 启动 v0 后端骨架（B 档），先把工具链基线（uv / ruff / pyright / pytest）落地，后续 task 才能跑测试。

## 思路
- 工具链全部按 plan 推荐：uv 包管理、ruff lint、pyright 类型检查、pytest + pytest-asyncio
- 依赖项严格按 plan，额外补 aiosqlite（plan 漏写）
- `pyproject.toml` 里 `tool.pytest.ini_options.asyncio_mode = "auto"` 让所有 `async def test_*` 自动当 async test

## 验收
- `cd backend && uv sync` 0 错误，`.venv/` 生成
- 关键依赖版本（实际安装）：
  - fastapi、uvicorn、sqlmodel 0.0.38、sqlalchemy 2.0.50、langgraph、openai、httpx 都按预期装好
  - aiosqlite 已加入并通过解析
