# T-013 — modules/ingest/repository

## 改了什么
- 新增 `backend/app/modules/ingest/repository.py`

## 为什么改
把数据访问封装在 repository 层，service 不直接拼 SQL（AGENTS.md R-1）。便于后续切换 ORM / 加缓存。

## 思路
- 4 个核心方法：create_document / find_document_by_sha256 / create_ingest_task / update_task_status，再加 list_tasks
- `update_task_status` 用 keyword-only 参数（`*` 之后），避免位置参数歧义
- 找不到 task 时抛 `ValueError`（暂时不引入 shared/exceptions，T-018 之前由 service 转 HTTP 400/404）
